kebab-case Converter: CSS Classes, URLs, and HTML Attributes

kebab-case uses lowercase letters with hyphens as word separators: user-profile, button-primary, max-width. It's the mandatory convention for CSS class names and custom properties, HTML data attributes (data-user-id), URL slugs (/blog/how-to-write-better), and REST API endpoint paths (/api/user-profile). Hyphens are also the standard separator in HTML attribute names.

This converter accepts any input format — plain text with spaces, camelCase, PascalCase, snake_case — and outputs clean kebab-case. Type "User Profile Settings" and get user-profile-settings. Type getUserProfileData and get get-user-profile-data. It removes special characters, normalizes whitespace, and lowercases everything in one pass.

Front-end developers use it constantly: generating class names from component names, creating URL slugs from article titles, writing CSS custom property names (--primary-button-color), and formatting HTML data attributes. It also helps when naming files and directories in web projects, where hyphen-separated names are preferred over spaces or underscores for cross-platform compatibility. The tool also shows all four case formats simultaneously so you can pick what fits each context.
