About Morse Code to Text Decoder

This free Morse Code to Text Converter decodes any valid Morse code string back into readable English. Enter your Morse code with spaces between each letter and a forward slash (/) or double space to separate words. The decoder supports all 26 letters, digits 0-9, and common punctuation. Morse code was invented by Samuel Morse and Alfred Vail in the 1830s and revolutionized long-distance communication. Today it is used in amateur radio (HAM radio), emergency signaling (SOS = ... --- ...), and accessibility technologies.

Frequently Asked Questions

What format should my Morse code be in?

Use dots (.) and dashes (-) with a single space between each letter code, and a forward slash (/) or double space between words. Example: .... . .-.. .-.. --- / .-- --- .-. .-.. -..

What does SOS look like in Morse code?

SOS is ... --- ... — three dots, three dashes, three dots. It is the international distress signal.
