Frequently Asked Questions

How do I convert HEX #5b5bd6 to RGB?
Split the value into hexadecimal pairs: 5B, 5B, and D6. In decimal they are 91, 91, and 214, so the CSS value is rgb(91, 91, 214).
What is the HSL value of #5B5BD6?
The HSL value is approximately hsl(240, 60%, 60%). Minor rounding differences can occur between conversion tools.
Can I pick and convert a different color?
Yes. Use the color input or select a preset swatch. The HEX, RGB, and HSL values update automatically and can be copied as valid CSS values.
