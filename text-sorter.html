Text Line Sorter

Paste multi-line text and sort lines alphabetically, by length, or numerically in any order.

About Text Line Sorter

The Text Line Sorter organizes multi-line text in alphabetical (A-Z or Z-A), length-based (shortest or longest first), numeric, or random order. A case-insensitive option ensures A and a are treated equally. Blank lines can be removed before sorting. It is ideal for sorting lists, keywords, file names, and data entries.

Frequently Asked Questions

Can I sort numbers with this tool?

Yes. Select "Numeric (ascending)" or "Numeric (descending)" to sort lines by their numeric value.

What happens to lines that are not numbers when numeric sort is used?

Non-numeric lines are treated as 0 (NaN parses to 0 in parseFloat comparison). For pure text data, use alphabetical sort.

Is this tool free?

Yes, completely free with no login required.
