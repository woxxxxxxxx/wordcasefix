About the Text Compare Tool

This free Text Compare tool performs a line-by-line diff between two texts, highlighting additions in green and deletions in red. It's ideal for comparing document versions, code snippets, configuration files, or any text where you need to quickly spot what changed. Options to ignore case and strip whitespace help reduce noise when comparing texts with minor formatting differences.

Frequently Asked Questions

How does the comparison work?

The tool performs a line-by-line comparison using the longest common subsequence (LCS) algorithm — the same technique used by git diff. Lines that exist only in the original are shown in red, lines only in the modified version in green, and unchanged lines in white.

Is this tool free?

Yes, completely free with no signup required. All processing happens in your browser.
