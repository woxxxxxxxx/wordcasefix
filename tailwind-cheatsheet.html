About Tailwind CSS Cheatsheet

This searchable Tailwind CSS cheatsheet covers all major utility classes including layout, flexbox, grid, spacing, sizing, typography, colors, borders, shadows, and animations. Frequently Asked Questions

Which version of Tailwind does this cover?

This cheatsheet covers Tailwind CSS v3 utility classes. Most classes are identical in v2, but v3 added JIT mode which allows arbitrary values like p-[13px].
